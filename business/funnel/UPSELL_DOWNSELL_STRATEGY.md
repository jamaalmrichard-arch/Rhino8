## ENHANCED FUNNEL ARCHITECTURE

```
FREE Book 1 → $7 Trilogy + $9 ORDER BUMP (audio kit)
  → OTO1: $47 Quick-Start Templates (20% take rate)
  → Downsell: $27 Mini Template (10% if OTO1 declined)
  → Questionnaire → Track Routing
  → Track Product ($97/$147/$197)
  → $497 Implementation Workshop (15% of track buyers)
  → $997 VIP Intensive (5% of track buyers)
  → Coaching Application → $5,997 Group Coaching
```

## PRODUCTS TO BUILD

### Order Bump — $9 Audio Trilogy Kit
- Audio version of all 3 trilogy books
- Checkbox on $7 checkout page
- Expected conversion: 35%
- Build time: 1 hour (create with AI voice or record)

### OTO1 — $47 Quick-Start Templates Pack
- Post-purchase page (after $7, before questionnaire)
- Financial calculator spreadsheet, budget template, net worth tracker, lender file template
- Expected conversion: 20%
- Build time: 2-3 hours

### Downsell — $27 Mini Template Pack
- Shown only if OTO1 declined
- Stripped version: just the budget template + calculator
- Expected conversion: 10%
- Build time: 1 hour

### $497 Implementation Workshop (4-Week Cohort)
- Live group calls, weekly for 4 weeks
- Track-specific implementation (apply what they learned)
- Hot seats, accountability, Q&A
- Offered via email to track product buyers
- Expected conversion: 15%
- Build time: 4-6 hours (sales page + registration)

### $997 VIP Intensive (Half-Day)
- Personalized deal analysis or financial plan review
- 1-on-1 video call, custom 90-day action plan
- Offered alongside or after $497 workshop
- Expected conversion: 5%
- Natural feeder to $5,997 coaching
- Build time: 3-4 hours (application page)
