# Due Diligence Checklist

## Before Making Offer
- [ ] Run comps (3+ sold, 3+ rental)
- [ ] Verify ARV with agent or MLS data
- [ ] Check tax records (county assessor)
- [ ] Check for liens (title search)
- [ ] Verify lot size and zoning
- [ ] Check flood zone (FEMA map)
- [ ] Drive by property (or Google Street View)
- [ ] Run 70% rule calculation
- [ ] Verify rental demand (Zillow, Rentometer, Craigslist)

## Under Contract (Inspection Period)
- [ ] Schedule property inspection
- [ ] Get contractor rehab estimate (walk the property)
- [ ] Verify utilities (water, sewer, electric, gas)
- [ ] Check for permits on previous work
- [ ] Verify roof age and condition
- [ ] Check HVAC age and condition
- [ ] Inspect foundation
- [ ] Check plumbing (water heater age, pipe material)
- [ ] Check electrical panel (amps, breaker type)
- [ ] Test for lead paint (pre-1978 homes)
- [ ] Verify square footage matches records
- [ ] Get insurance quote
- [ ] Order title search / title commitment
- [ ] Verify HOA restrictions (if applicable)
- [ ] Check for environmental issues
- [ ] Verify property boundaries / survey

## Before Closing
- [ ] Final walk-through
- [ ] Review closing disclosure / settlement statement
- [ ] Verify all liens cleared
- [ ] Confirm funding with lender
- [ ] Set up insurance policy
- [ ] Arrange contractor start date (BRRRR/Flip)
- [ ] Set up property management (BRRRR)
- [ ] File entity docs if buying in LLC

## After Closing
- [ ] Record deed
- [ ] Start rehab (get permits if needed)
- [ ] Set up utility accounts
- [ ] List for rent (BRRRR) or list for sale (Flip)
- [ ] Update portfolio tracker
- [ ] Save all receipts for tax purposes
