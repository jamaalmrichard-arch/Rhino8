## ON-PAGE SEO CHECKLIST

Apply to every blog post and landing page:

### Title Tag
- [ ] Primary keyword included naturally
- [ ] Under 60 characters
- [ ] Includes a benefit or curiosity element
- [ ] Brand name at end if space allows: "... | J. Marque"

### Meta Description
- [ ] 150-155 characters
- [ ] Primary keyword included
- [ ] Compelling reason to click (benefit or hook)
- [ ] Not a duplicate of any other page

### URL Structure
- [ ] Short and clean: `/history-of-money` not `/the-complete-history-of-money-from-gold-to-digital-2026`
- [ ] Primary keyword in URL
- [ ] Hyphens between words, no underscores
- [ ] Lowercase only

### Headers
- [ ] H1: One per page, includes primary keyword
- [ ] H2s: Include secondary keywords naturally
- [ ] H3s: Support H2 sections, use related terms
- [ ] Logical hierarchy (no skipping from H2 to H4)

### Content
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword density: 1-2% (natural, not stuffed)
- [ ] Secondary keywords sprinkled throughout
- [ ] Internal links: 3-5 per article (to other blog posts + landing pages)
- [ ] External links: 2-3 per article (authoritative sources)
- [ ] Images with keyword-rich alt text
- [ ] Minimum 1,500 words (longer articles rank better)
- [ ] Table of contents for posts over 2,000 words

### Technical
- [ ] Page loads in under 3 seconds
- [ ] Mobile-responsive
- [ ] Schema markup (Article, FAQ, or HowTo as appropriate)
- [ ] Open Graph tags for social sharing
- [ ] Canonical tag set correctly
- [ ] No broken links

### CTA Integration
- [ ] Free book opt-in CTA in introduction
- [ ] In-content CTA at the midpoint
- [ ] End-of-article CTA with opt-in form
- [ ] Exit-intent popup (if available)

## LINK BUILDING STRATEGY

### Phase 1: Foundation (Months 1-3)
- **Guest Posting:** Write for financial literacy blogs, RE investing sites, veteran-focused publications. Include J. Marque bio with link to free book.
- **HARO / Connectively:** Respond to journalist queries about real estate investing, financial literacy, money history, veteran finance. Earn links from high-DA publications.
- **Podcast Appearances:** Pitch J. Marque (via AI avatar or voice) to financial literacy, RE investing, and veteran podcasts. Show notes = backlinks.
- **Resource Pages:** Find "best financial literacy resources" pages and pitch Money Decoded for inclusion.

### Phase 2: Growth (Months 4-6)
- **Original Data/Research:** Create shareable statistics (e.g., "We analyzed 50 years of savings account returns vs inflation" infographic). Data gets linked to naturally.
- **Broken Link Building:** Find broken links on financial education sites, offer Money Decoded content as replacement.
- **Community Participation:** Quora answers, Reddit (r/personalfinance, r/realestateinvesting, r/Fire), financial forums. Provide value, link where relevant and allowed.
- **Collaborative Content:** Co-create content with other financial educators for mutual linking.

### Phase 3: Authority (Months 7-12)
- **Digital PR:** Pitch stories around financial literacy statistics, inflation impact studies, or RE market analysis.
- **Tool/Calculator Creation:** Build free tools (inflation calculator, rental property analyzer) that earn natural backlinks.
- **Scholarship/Grant:** Create a small financial literacy scholarship/grant for veterans -- earns .edu links.
- **Industry Roundups:** Create "50 Financial Literacy Experts Share Their Best Advice" style content. Everyone linked shares it.

### Link Building Targets
| Source Type | DA Range | Links/Month Goal | Method |
|------------|---------|-----------------|--------|
| Guest posts | 30-60 | 2-4 | Outreach |
| HARO/media mentions | 50-90 | 1-2 | Reactive |

- Include primary keyword naturally in the first 40 characters

```
...

FREE BOOK: [opt-in page URL]
FOLLOW J. MARQUE:
- Instagram: [URL]
- TikTok: [URL]
- Twitter/X: [URL]

#FinancialLiteracy #MoneyDecoded #[keyword hashtag]
```

### Tags (15-20 per video)
- Primary keyword as first tag
- Include exact title as a tag
- Mix of broad and specific: "financial literacy" + "what happened in 1971 nixon shock"
- Include competitor channel names as tags (ethical targeting)
- Include common misspellings of niche terms

### Thumbnail Best Practices
- Gold (#c8a84e) and black (#0a0a0a) brand colors
- 3-4 words maximum of text
- High contrast, readable at mobile size
- Expressive face (AI avatar) showing emotion relevant to topic
- Consistent template style for brand recognition

### YouTube Shorts SEO
- Title: Include keyword + "#Shorts" tag
- Description: 2-3 lines + free book link + 3 hashtags
- Add to relevant playlists
- Cross-promote in community tab
